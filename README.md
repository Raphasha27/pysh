# Pysh

A Python-powered shell alternative with smart autocomplete, syntax highlighting, and command history search.

## Features

- autocomplete
- cli
- python
- shell
- terminal
- CLI

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Raphasha27/pysh.git
cd pysh
```
